# Portfolio Website

This portfolio website is built with HTML, CSS, and JavaScript.

## Files
- `index.html` — the main landing page
- `styles.css` — styling for the page
- `script.js` — mobile menu and contact form interactions
- `assets/` — place your profile image here

## How to use
1. Place your profile photo at `assets/profile.jpg`.
2. Open `index.html` in a browser.
3. Edit the text, email, and phone details in `index.html` to match your profile.

## Notes
- The page includes Home, About, and Contact sections.
- It is responsive for mobile and desktop.
- The contact form has a simple success message when submitted.
